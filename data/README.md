# Datasets used in the course

- [`IRIS.csv`](./IRIS.csv): Fisher's iris dataset from Fisher, R. A. “The Use of Multiple Measurements in Taxonomic Problems.” Annals of Eugenics 7, no. 2 (1936): 179–88. https://doi.org/10.1111/j.1469-1809.1936.tb02137.x.
   - Tasks: classification, clustering.
- [`Housing.csv`](./Housing.csv): housing prices dataset from https://www.kaggle.com/datasets/yasserh/housing-prices-dataset.
   - Tasks: regression.
- [`mnist_train.parquet`](./mnist_train.parquet) and [`mnist_test.parquet`](./mnist_test.parquet): MNIST dataset from https://huggingface.co/datasets/ylecun/mnist converted to `.parquet` format.
   - Tasks: classification.
